- ハッシュタグ（IG基本セット・10個前後）: #麻雀 #麻雀好きと繋がりたい #麻雀好き #セット麻雀 #麻雀アプリ #三麻 #四麻 #麻雀初心者 #麻雀サークル #まじゃすこ（Xは0〜2個まで）

## Instagramストーリーズ（サイズ1080×1920・24時間で消える。使えるものは「ハイライト」に保存）
ストーリーズは**IGで唯一リンクを貼れる場所**（リンクスタンプ）。フィード投稿と役割を分ける:
フィード=ストック（作り込んだ画像）、ストーリーズ=フロー（気軽・参加型・誘導）。
1. **新規投稿の再シェア** — フィード投稿を出したら必ずストーリーズでもシェア（「新しい投稿🀄」+タップ誘導）。工数ほぼゼロで基本の型
2. **リンク誘導** — アプリ（majasco.jp）や新ブログ記事へリンクスタンプで直行導線。「今夜のセット麻雀に👇」など一言添える
3. **アンケート/クイズスタンプ** — 参加型で接点を作る。ネタ例: 「三麻派？四麻派？」「ウマは10-30派？5-10派？」
   「Q. 30符4翻は何点？」（クイズ）「焼き鳥ルール入れる？」「チップあり派？」
4. **質問スタンプ** — 「麻雀のスコア計算で分からないこと、答えます」→ 回答をストーリーズ+ブログ記事のネタに還元
5. **ミニTips** — アプリの小ワザを1枚1機能で（例: 「点数は下2桁入力不要」「最後の1人は自動計算」）。feature画像の流用OK
6. **アップデート速報** — /news/記事のリンクスタンプ+ひとこと。X告知と同時に
- **ハイライト構成**: 「使い方」（howtoカルーセル）／「機能」（feature+Tips）／「Q&A」（質問回答）／「お知らせ」
- 頻度目安: 週2〜3本（フィード投稿の再シェア含む）。文字は中央帯に置く（上下は時計・リアクションUIと被る）
- 用語ルール・投稿前QAはフィードと共通。クイズの答えに符計算が絡むときは「点数計算はアプリ対象外」の線引きに注意（アプリ誘導は「集計はまじゃすこで」まで）

## 制作手順（画像つき投稿）
1. `tools/` のスクリプトで生成（新パターンは既存スクリプトをコピーして作る）
2. 出力先は `まじゃすこ素材/`（コミットされない）。IGシリーズは `まじゃすこ素材/ig/`
3. **生成したらClaude自身がReadで画像を開き、下のQAチェックリストを通してから**ユーザーに提示
4. 投稿文はこのdocのテンプレ・過去文面（git履歴/会話）を踏襲

## 投稿前QAチェックリスト
- [ ] 画像内の文字がスマホ枠・カード枠・画面の内側に収まっている
- [ ] 装飾（✨・ドット）が文字と重なっていない
- [ ] 用語ルール違反がない
- [ ] majasco.jp の表記がある
- [ ] サイズ: IG=1080×1080 ／ Xヘッダー=1500×500（左下アイコン・上下60px安全域）／ OGP=2:1トリミング考慮
- [ ] 文面: Xは末尾にURL／IGはプロフィール誘導・ハッシュタグ

## 落とし穴（トラブルの記憶。修正したらここに追記）
- **Xのカードキャッシュ**: 同一URLは約1週間キャッシュ。テスト投稿は `?v=N` を変える。OGP画像の差し替えは**ファイル名ごと変更**（ogp.png→ogp2.png方式）＋全ページのmeta書き換え
- **Xカードにタイトル文は出ない**（画像＋ドメインのみ。2023年仕様）
- **画像内文字のはみ出し**: PILは自動折返ししない。2026.07.11にhowtoカルーセルで発生（チップ幅・チャット文・合計バー）→ 生成後の目視QAを必須化
- **カラー絵文字の描画**: seguiemj.ttf + `embedded_color=True` でPILから描画可能（🥇🥈🥉実績）
- **装飾の重なり**: スパークルは文字の行末付近（右端）に置かない
